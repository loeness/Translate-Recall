# Changelog

本项目的所有重要变更都会记录在此文件中。

格式参考 Keep a Changelog，并遵循语义化版本（SemVer）。

## [1.0] - 2026-03-27

### Added

- 支持对翻译后页面进行原文预处理和映射。
- 支持点击单句展示对应原文。
- 支持划词后聚合展示原文片段。
- 新增生产级目录结构（assets/src/_locales）。
- 新增多语言扩展元信息（en 与 zh_CN）。
- 新增扩展图标资源（16/48/128）。
- 新增 README、LICENSE、发布模板与 Git 忽略规则。

### Changed

- 将 popup 样式从 HTML 内联拆分到独立文件。
- 调整 manifest 路径引用以匹配新目录布局。

### Fixed

- 修复内容脚本未注入时的消息连接失败问题（自动注入兜底）。
- 修复 Tooltip 原文可能被浏览器翻译器二次翻译的问题。
